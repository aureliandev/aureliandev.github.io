NetSuite JSON API

NetSuite's RESTlets provide developers with a powerful tool for integrating external systems with NetSuite. Currently, each RESTlet is custom-built for a specific business need, meaning that the inputs and outputs vary by RESTlet.

We believe there is a great opportunity to build a structured, well-documented JSON API for interacting with NetSuite records through RESTlets. This would provide developers a common JSON interface to work with, making development faster and more repeatable.

Key Advantages

  • Consistent, repeatable integrations can be leveraged across multiple client accounts
  • Lightweight JSON API
  • Publicly-available API documentation
  • Easy-to-install SuiteBundle available on SuiteApp.com
